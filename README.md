**CS @ Stanford**  

I like computer systems, and I get to study them at Stanford University!

Here are some of my recent projects:
- [LoDAR](https://github.com/ryanmccormickdev/lodar): As a final project for CS107E (Bare-Metal Computer Systems from the ground up), I built a RISCV bare-metal Lo-Fi ToF LiDAR scanning system entirely from scratch in C. This was done with my partner Manit Mehta, who helped with graphics and rendering.
- [Vibeathons](https://github.com/ryanmccormickdev/vibeathons/blob/main/vibeathon.pdf): As an extra credit opportunity in CS109 (Probabiliity for CS), I did some webscraping and analysis of [Treehacks](https://treehacks.com/) repositories. I mapped how agents have changed the programming landscape and how we'll see the developer mindset change in the next few years.
- [Pocketty](https://github.com/tnvrdy/pocketty): My Treehacks 2026 project! Built with my good friends Tanvi Reddy and Manit Mehta, pocketty is a [Pocket Operator](https://teenage.engineering/products/po)-style sequencer and sampler that runs cross-platform in the terminal with an adorable TUI. I was in charge of the audio/sequencing/sampling backend as well as the API design. This was our first project written entirely in rust.
